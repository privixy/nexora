# Nexora

Nexora is a lightweight, developer-focused database client built with Tauri, Rust and React.

## Key Features

- Native drivers for PostgreSQL, MySQL/MariaDB and SQLite
- External database driver plugins over JSON-RPC stdio
- Connection profiles with local storage and OS keychain support
- SSH tunneling
- Monaco SQL editor with tabs, autocomplete and query history
- SQL notebooks with Markdown cells and charts
- Visual query builder and Visual EXPLAIN
- Data grid with inline editing and export
- Optional AI assistant and MCP server

## Development

```bash
pnpm install
pnpm tauri dev
```
