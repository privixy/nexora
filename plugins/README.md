# Nexora Plugins

Nexora supports external database driver plugins over JSON-RPC 2.0 via stdin/stdout.

## Registry

The bundled registry is stored in [`registry.json`](./registry.json). It is empty after the rebrand and can be populated with Nexora-owned or trusted plugin entries later.
